
Detecting whether a person in a video stream(real time) is wearing a mask or not using python,OpenCV and keras.This project is very useful in pandemic situation.It can be used in malls,traffic places so that we can identify who are wearing masks and who aren't wearing mask.
The packages used  for this project are:
tensorflow>=1.15.2
keras==2.3.1
imutils==0.5.3
numpy==1.18.2
opencv-python==4.2.0.*
matplotlib==3.2.1
scipy==1.4.1
